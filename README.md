# AutoFeedback
## 如何使用
1. 双击Main.exe运行程序
2. 在程序中输入用户名、密码
3. 选择当天知识的掌握程度，可多选，多选就会从选项中随机选取一个
4. 可选输入建议或建议
5. 可选是否匿名反馈
6. 点击自动反馈按钮，程序将自动完成反馈。可在按钮上方看见程序运行进度
